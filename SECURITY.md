# Security Policy

## About This Repository

This repository is an **academic reference implementation** from a PhD thesis on Interactive Explainable AI Systems. It is not actively maintained or deployed as a production service.

## Known Dependency Issues

This project uses TensorFlow 2.3.0 and associated ML/XAI libraries (SHAP, LIME, DiCE) from 2020-2021. These versions contain known vulnerabilities but cannot be updated without breaking the reference implementation and its compatibility with the thesis results.

**Do not use this code in production without updating all dependencies.**

## Reporting

If you discover a security issue specific to the code in this repository (not in its third-party dependencies), please open an issue.
